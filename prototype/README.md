# 陌恋 App 高保真原型

## 📱 项目概述

这是"陌恋"笔友社交分享 App 的高保真 HTML 原型，完全按照 iOS 设计规范和现代 UI 设计理念打造。

## 🎨 设计特点

### 视觉设计
- **优雅的渐变配色**：使用 #9D31FF → #F260FF → #FF609F 的品牌渐变色
- **清新的背景色系**：浅灰紫 #F8F9FD 营造舒适的视觉体验
- **精致的圆角设计**：12px 标准圆角，20px 大圆角，营造柔和感
- **细腻的阴影系统**：使用品牌色阴影，增强层次感

### 交互设计
- **流畅的页面切换**：底部导航支持三个主页面切换
- **点击反馈动画**：所有按钮都有 scale 缩放反馈
- **双击点赞功能**：双击卡片快速点赞
- **点赞粒子效果**：点赞时的爱心粒子动画

### 功能亮点
- **动态广场**：社交分享，支持图片和文字
- **时光笔记**：时间轴式私密日记
- **个人中心**：数据展示和功能入口
- **情绪标签**：为笔记添加情绪状态
- **氛围选择**：为动态选择氛围标签

## 🚀 使用方法

### 1. 直接在浏览器中打开

```bash
# 进入原型目录
cd molian/prototype

# 使用浏览器打开任意 HTML 文件
open index.html          # macOS
start index.html         # Windows
xdg-open index.html      # Linux
```

### 2. 使用本地服务器（推荐）

```bash
# 使用 Python 启动简单服务器
cd molian/prototype
python3 -m http.server 8000

# 然后在浏览器访问
# http://localhost:8000/index.html
```

### 3. 使用 VS Code Live Server

1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 📱 页面功能

### index.html - 主应用界面

**动态页面**
- 查看用户发布的动态
- 点赞、评论、更多操作
- 关注用户
- 悬浮发布按钮
- 双击卡片快速点赞

**笔记页面**
- 时间轴展示笔记
- 查看笔记详情
- 删除笔记
- 渐变色时间轴设计

**我的页面**
- 个人信息展示
- 粉丝/关注/获赞数据
- 最近动态横向滚动
- 功能菜单入口

**底部导航**
- 三个主页面切换
- 激活状态渐变色显示

### publish.html - 发布动态

- 分类选择（美食/工作/学习/宠物/心情）
- 文字输入
- 图片上传（最多3张）
- 氛围标签选择
- 位置添加

### note-edit.html - 编辑笔记

- 标题和正文输入
- 情绪选择（6种情绪）
- 图片上传（最多3张）
- 天气记录
- 标签管理
- 富文本工具栏

## 🎯 设计规范

### 颜色系统

**品牌色**
```css
主渐变: linear-gradient(135deg, #9D31FF 0%, #F260FF 50%, #FF609F 100%)
主紫色: #9D31FF
亮紫色: #F260FF
粉红色: #FF609F
```

**背景色**
```css
页面底色: #F8F9FD
卡片背景: #FFFFFF
输入框背景: #FAFBFF
```

**文字色**
```css
主文字: #1A1A2E
次文字: #6B7280
弱文字: #9CA3AF
```

### 间距系统

```
4px   - 极小间距
8px   - 小间距
12px  - 中小间距
16px  - 标准间距
20px  - 中等间距
24px  - 大间距
32px  - 超大间距
```

### 圆角系统

```
8px   - 小元素（按钮、输入框）
12px  - 标准卡片
16px  - 大卡片
20px  - 底部导航
50%   - 圆形（头像、圆形按钮）
```

### 阴影系统

```css
/* 标准卡片 */
box-shadow: 0 2px 8px rgba(157, 49, 255, 0.08);

/* 悬浮元素 */
box-shadow: 0 4px 16px rgba(157, 49, 255, 0.12);
```

## 📝 开发建议

### 转换为 Flutter 应用

1. **布局结构**
   - 使用 `Scaffold` 作为页面框架
   - `BottomNavigationBar` 实现底部导航
   - `ListView` 实现滚动列表

2. **样式实现**
   - 使用 `LinearGradient` 实现渐变色
   - `BoxDecoration` 实现圆角和阴影
   - `Theme` 统一管理颜色和字体

3. **交互实现**
   - `GestureDetector` 实现点击、双击、长按
   - `AnimatedContainer` 实现动画效果
   - `Hero` 实现页面转场动画

4. **状态管理**
   - 使用 Provider 或 Riverpod
   - 管理点赞、关注等状态
