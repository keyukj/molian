# 探友 - 笔友社交分享 App 产品文档

## 一、产品概述

### 1.1 产品定位
探友是一款融合社交动态与个人笔记的轻量级社交应用，专注于真实生活记录与情感表达的分享平台。

### 1.2 核心价值
- **双重记录模式**：公开社交动态 + 私密个人笔记，满足用户不同场景的记录需求
- **情感化设计**：渐变紫粉色系营造温暖、浪漫的社交氛围
- **轻量化交互**：简洁的三页式结构，降低用户认知负担

### 1.3 目标用户
- 18-35岁年轻用户群体
- 喜欢记录生活、分享日常的文艺青年
- 追求情感共鸣与真实社交的用户

## 二、功能架构

### 2.1 核心功能模块

#### 模块一：动态广场（公开社交）
**功能定位**：用户生活分享的公共空间

**核心功能**：
1. **动态发布**
   - 支持类型：美食、工作、学习、宠物、心情
   - 内容形式：文字 + 图片（1-3张）或纯文字
   - 发布入口：悬浮按钮（右下角）

2. **动态展示**
   - 时间流排序（最新在前）
   - 卡片式布局
   - 信息元素：
     * 用户头像（可点击进入主页）
     * 昵称
     * 关注按钮（未关注显示，已关注隐藏）
     * 发布时间（智能显示：刚刚/分钟前/小时前/日期）
     * 动态内容（文字 + 图片网格）
     * 互动区：点赞数、评论数、更多操作

3. **互动功能**
   - 点赞：双击卡片或点击爱心图标
   - 评论：展开评论列表，支持回复
   - 更多操作：举报、拉黑、屏蔽（三点菜单）

4. **独特交互**
   - **下拉刷新**：渐变色加载动画
   - **长按预览**：长按动态卡片快速预览大图
   - **滑动操作**：左滑快速点赞，右滑收藏

#### 模块二：时光笔记（私密记录）
**功能定位**：个人专属的时间轴日记本

**核心功能**：
1. **笔记创建**
   - 快速记录入口（顶部 + 按钮）
   - 支持：标题、正文、图片（最多3张）
   - 自动保存草稿

2. **时间轴展示**
   - 左侧：垂直时间轴（年-月-日）
   - 右侧：笔记卡片
   - 卡片内容：
     * 标题（加粗显示）
     * 正文预览（最多3行）
     * 图片网格（最多3张）
     * 删除按钮（右上角）

3. **笔记管理**
   - 点击卡片：展开完整内容
   - 编辑：长按卡片进入编辑模式
   - 删除：确认弹窗防误删

4. **独特设计**
   - **时间节点标记**：重要日期用渐变色圆点标注
   - **情绪标签**：可为笔记添加情绪图标（开心/难过/平静等）
   - **天气记录**：自动记录当天天气

#### 模块三：个人中心
**功能定位**：用户信息管理与数据展示

**核心功能**：
1. **个人信息展示**
   - 大头像（支持点击更换）
   - 昵称（可编辑）
   - 个性签名

2. **社交数据**
   - 粉丝数（点击查看粉丝列表）
   - 关注数（点击查看关注列表）
   - 获赞数（总点赞统计）

3. **最近动态**
   - 展示最近3条发布的动态
   - 横向滑动卡片

4. **功能入口**
   - 设置：账号安全、隐私设置、通知管理
   - 我的收藏
   - 浏览历史
   - 关于我们

### 2.2 独特功能亮点

1. **情绪日历**
   - 在笔记页面顶部展示月度情绪热力图
   - 用渐变色深浅表示每日情绪状态

2. **陌生人信箱**
   - 随机匹配功能，向陌生人发送匿名明信片
   - 增强社交趣味性和神秘感

3. **回忆闪现**
   - 智能推送一年前/半年前的今天的笔记
   - 增强情感连接

4. **动态氛围标签**
   - 发布动态时可选择氛围标签（治愈/元气/emo/chill）
   - 用户可按氛围筛选动态

## 四、操作路径设计

### 4.1 核心路径

**路径1：发布动态**
```
动态页 → 点击悬浮按钮 → 选择类型 → 输入内容 → 添加图片（可选）→ 发布
```

**路径2：互动动态**
```
动态页 → 双击卡片点赞 / 点击评论图标 → 输入评论 → 发送
```

**路径3：创建笔记**
```
笔记页 → 点击顶部+按钮 → 输入标题和正文 → 添加图片 → 自动保存
```

**路径4：查看他人主页**
```
动态页 → 点击用户头像 → 进入主页 → 查看TA的动态 → 关注/返回
```

### 4.2 独特交互路径

**快速点赞**：动态卡片左滑 → 显示爱心图标 → 完成点赞

**快速收藏**：动态卡片右滑 → 显示星标图标 → 完成收藏

**长按预览**：长按动态图片 → 全屏预览 → 松手返回

## 五、技术要求

### 5.1 性能指标
- 首屏加载时间 < 1.5s
- 动态列表滑动帧率 ≥ 60fps
- 图片加载采用渐进式加载

### 5.2 兼容性
- iOS 14.0+
- 支持 iPhone SE 至 iPhone 15 Pro Max

### 5.3 数据存储
- 笔记本地加密存储
- 动态云端同步
- 图片 CDN 加速
